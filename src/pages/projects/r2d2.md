### Project Overview
Built as part of Worcester Polytechnic Institute's Unified Robotics II: Sensing (RBE 2002) course, "R2-D2" is a differential-drive autonomous mobile robot engineered for complex path planning, maze navigation, and precise object interaction. The system leverages a robust C++ control architecture to continuously fuse data from multiple analog and digital sensors, allowing it to dynamically track its environment and execute high-level navigational tasks.

To achieve this, the software utilizes a custom Forward and Inverse Kinematics (FK/IK) engine based on second-order approximations to translate wheel odometry and IMU gyroscope data into a global Cartesian coordinate frame. By utilizing proportional-integral (PI) control loops and advanced trajectory states, the robot can autonomously drive to specific *(X, Y, θ)* poses, align with target markers, and track precise grid intersections without human intervention.

### Hardware Setup
The perceptual and mechanical architecture of R2-D2 pairs an agile differential-drive chassis with a deep stack of onboard instrumentation to create a highly responsive, closed-loop navigation system.

* **Locomotion & Odometry:** Built on a Pololu Romi 32U4 chassis featuring magnetic quadrature encoders. The drivetrain uses custom PI speed controllers to ensure perfectly straight trajectories, while utilizing an onboard IMU to execute highly accurate 90-degree in-place rotations.

* **Vision & Maze Navigation:** Equipped with an OpenMV camera module for real-time AprilTag detection. The robot uses visual distance approximation and tag ID parity (even/odd) to autonomously make routing decisions, effectively solving physical mazes and halting at designated exit tags.

* **Environmental Sensors:** Integrates both HC-SR04 ultrasonic rangefinders and Sharp IR sensors. To prevent erratic control oscillation (jittering) during object approach, the system utilizes advanced signal conditioning, implementing both software hysteresis and a hardware Schmitt trigger. Multi-element reflectance arrays are also utilized for autonomous line-following and intersection tracking.

* **Custom Instrumentation:** Features a custom-built digital scale to weigh payload objects. The sensor stack includes a strain gauge load cell wired into a Wheatstone bridge, which is amplified by an AD620 instrumentation amplifier and digitized by an HX711 ADC. The software applies linear regression and statistical noise filtering (averaging) to calculate payload mass with high precision.
